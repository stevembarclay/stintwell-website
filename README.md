# stintwell-website

## Design System

This project uses a systematic design system. See [docs/design-system.md](docs/design-system.md) for guidelines.

### Key Principles

- Typography: use utility classes (`.text-label`, `.text-h2`, etc.) instead of manual Tailwind.
- Cards: use `.card`, `.card-hover`, and `.card-glow` utilities for consistent styling.
- Components: use primitives in `components/primitives/` (`Button`, `Input`, `Select`, `Icon`).
- Accessibility: include `.focus-ring` and appropriate ARIA attributes on interactive elements.
